# fsdevtools-cli-api — Public API Contracts

Defines the stable interfaces (`Command`, `Config`, `Result`, `ScriptEngine`) that all commands and plugins must implement. This is the contract between the CLI core and all command/plugin modules.

## Reference Files

| File | When to read |
|---|---|
| `AGENTS-CONTRACTS.md` | Implementing a new command, config, result type, or script engine plugin |
| `AGENTS-BUILD.md` | Build setup and transitive dependency exposure (Airline, Jackson) |
