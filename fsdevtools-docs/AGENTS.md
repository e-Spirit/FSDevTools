# fsdevtools-docs — Interactive Documentation UI

Vue.js web application providing a searchable, interactive UI for all CLI commands. Consumes `data.json` from `fsdevtools-docs-generator`. Node.js 12.18.1 is downloaded automatically — no local install needed.
