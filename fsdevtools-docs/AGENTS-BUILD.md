# Build, Input & Output

## Build

```bash
# Full build (installs Node, runs npm install, builds Vue app)
./gradlew :fsdevtools-docs:buildVueApp

# Serve locally for development
./gradlew :fsdevtools-docs:serveVueApp

# Clean build output
./gradlew :fsdevtools-docs:clean
```

The `buildVueApp` task depends on:
1. `npmInstall` — installs npm packages
2. `createDocumentationJson` (root task) — generates `build/assets/data.json`

## Input Files

| File | Source |
|---|---|
| `build/assets/data.json` | Generated by `fsdevtools-docs-generator` via root task `createDocumentationJson` |
| `build/assets/application.json` | Generated by root task `writeArtifactInfo` (version, branch, commit) |

## Output

- `dist/` — compiled Vue.js application (static HTML/JS/CSS)
- Included in distribution archives at `fs-cli/docs/`

## Development Workflow

```bash
# Rebuild the JSON data first
./gradlew createDocumentationJson

# Then serve the UI locally
./gradlew :fsdevtools-docs:serveVueApp
```
