# fsdevtools-commands — Command Group Aggregator

No source code of its own — aggregates all command group submodules so `fsdevtools-cli` gets every command via a single dependency.

## Reference Files

| File | When to read |
|---|---|
| `AGENTS-STRUCTURE.md` | Finding which submodule owns a group, or understanding the shared code pattern all submodules follow |
| `AGENTS-BUILD.md` | Building command modules or adding a new command group |
