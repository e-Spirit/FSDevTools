# fsdevtools-scriptengines:Javascript — JavaScript (Nashorn) Script Engine Plugin

Adds OpenJDK Nashorn JavaScript execution to `script run` / `script parse`. Engine name: `javascript`. Nashorn ships as a standalone library since Java 15 removed it from the JDK.
