# fsdevtools-scriptengines:Groovy — Groovy Script Engine Plugin

Adds Apache Groovy (JSR-223) script execution to `script run` / `script parse`. Engine name: `groovy`. Distributed as a standalone fat JAR for the `plugins/` directory.
