# fsdevtools-scriptengines — Script Engine Plugins (Aggregator)

Aggregates optional script engine plugin fat JARs (Groovy, JavaScript) that users drop into the `plugins/` directory to extend `script run` / `script parse` beyond built-in BeanShell.

## Reference Files

| File | When to read |
|---|---|
| `AGENTS-PLUGINS.md` | Adding a new engine plugin, understanding distribution model, or classloader constraints |
| `AGENTS-BUILD.md` | Building or publishing the engine fat JARs |
