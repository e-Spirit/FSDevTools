This file provides guidance to AI-Assistants/LLMs when working with code in this repository.

**Key versions:** Java `17`, Gradle `8.13` (Kotlin DSL), FirstSpirit API `5.2.231105`, License: Apache 2.0, Group: `com.espirit.moddev.fsdevtools`

## Reference Files

Read the relevant file before starting work in that area.

| File | When to read |
|---|---|
| `AGENTS-ARCHITECTURE.md` | Navigating modules, understanding inter-module dependencies, or identifying which subproject to edit |
| `AGENTS-BUILD.md` | Running builds, tests, or publishing; understanding versioning, dependency catalog, or CI/CD flow |
| `AGENTS-CLASSLOADING.md` | Adding or modifying dependencies, writing plugin JARs, or dealing with classpath/scope issues |
| `AGENTS-COMMAND-AUTHORING.md` | Adding a new command or command group, or tracing the command execution flow |
| `AGENTS-RUNTIME.md` | Understanding the distribution layout or environment variable defaults |
| `AGENTS-TRAPS.md` | Before making any structural change — lists known pitfalls that cause incorrect edits |
